┌─────────────────────────────────────────────────────────────────┐
│  🔧 TROUBLESHOOTING                                             │
└─────────────────────────────────────────────────────────────────┘

⚠ Port 3000 already in use?
   → Create .env file with: PORT=3001
   → Then run: npm start

⚠ npm not found?
   → Install Node.js from https://nodejs.org/

⚠ Permission errors?
   → Run PowerShell as Administrator

⚠ Firewall blocking?
   → Allow Node.js in Windows Firewall
